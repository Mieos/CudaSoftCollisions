# CudaSoftCollisions
CudaSoftCollisions is a library that allow you to detect self collisions in a tetrahedrical mesh that is deformed over time.

## Build the project
* Clone the project :
```bash
git clone https://github.com/Mieos/CudaSoftCollisions.git
```
* Build the project :
```bash
mkdir build 
cd build
cmake ..
make
```
